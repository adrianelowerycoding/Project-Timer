GUI Notes/Plan:

Before I can tackle the logic of my program, I need to create its GUI with Tkinter. I will explain in great detail the main window now: 

This project is meant to allow users to click boxes timers that will track time when clicked and stop  tracking when clicked again. I want the main window to be split into 3 sections: 

Section 1: A button and a drop down menu. Button: A "Create Initial database" button. Drop down menu: A small drop down called "Options" that drops down and reveals "Update database" option and "View Day's data" option. 

Section 2: Two buttons. Button 1: "Create Project Timer", Button 2: "Delete Timer"

Section 3: A box for the created timers. These timers will each be their own small box that contain a title <Project number> and a round button. The round button will say "Start" when not clicked initially and "Stop" after being clicked initially. Think of Microsoft's clock app.
